Prgressive Enhancement

Progressive Enhancement is a powerful methodology that allows Web developers to focus on creating the finest possible websites while balancing the challenges that come with having such websites viewed by a variety of unknown user-agents.It focuses on content( reason we create websites to begin with).

Importance

    It allows everyone to access the basic content
    It provides enhanced version of the page to those with more advanced browser software.
    It speeds up the interaction for some of the user
    Search speed optimization
